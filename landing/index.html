What it does

Built for clarity — abstract numbers become visual feedback you can act on immediately.

📊

Dual dashboard cards

Overall budget + remaining on the left; on-track progress % on the right with green, gold, or red status.

💵

Animated cash stacks

3D bill artwork shakes and flies off when a charge hits — the closest denomination stack animates away.

📅

Timeframe preview slider

Scrub through days in your budget period to preview how your on-track score would look without changing real data.

🔄

Multi-budget carousel

Swipe between named budgets — Food, Transport, General — each with its own timeframe and transactions.

💳

Bank notification demo

Simulate incoming Coffee Corner charges ($1–$40) with a full deduction animation and recent activity updates.

☁️

Sign in & sync

JWT auth with Express API — create an account, sign in, and sync budgets across devices.

On-track progress, explained

Answers one question: “Am I spending at a sustainable pace for the time left?”

expectedRemaining = budget × (daysRemaining / totalDays)
onTrackProgress   = (remaining / expectedRemaining) × 100
  • ≥ 75% Budget Master!
  • 50–74% Still On Track!
  • < 50% Budget Better!

Architecture

Local-first mobile app with optional cloud sync via REST API.

Mobile (Expo)

  • React Native 0.85 + React 19
  • React Navigation 7
  • BudgetContext + AuthContext
  • AsyncStorage (offline cache)
↔

API (Node.js)

  • Express REST server
  • JWT authentication
  • Per-user budget sync
  • JSON file persistence
